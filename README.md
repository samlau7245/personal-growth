# 驾驶经验

* **遇见并线，果断决定。** 要么减速让行，要么坚决不允许，不能模棱两可不然会给人误解。
* **保持车距，安全第一。** 城市道路遵循3秒法则，高速遵循4秒法则。
* **变道，先看后视镜再打转向灯。**
* **高速紧急，只踩刹车，不打方向盘。**
* **刹车先看后视镜。** 松开一点刹车的力，防止后车追尾。
* **看远顾进。**
* **倒车时刻保持警惕。**
* **车内音响不要太大声。**
* **遇到故障：车靠边、人撤离、即报警。** 高速150外放三角牌。
* **路边停车，提前观察。**
